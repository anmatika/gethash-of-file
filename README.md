# USAGE

## Prequisites
`npm i`

## Output all the algorithms
`node index.js filename`

## Output with command line options

### Examples

Get sha1
`node index.js filename --sha1`

Get sha256
`node index.js filename --sha256`

Get md5
`node index.js filename --md5`

Get md5 and sha1
`node index.js filename --md5 --sha1`
